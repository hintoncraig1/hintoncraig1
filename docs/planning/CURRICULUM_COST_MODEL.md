# Week 4 Curriculum + Cost Model

## Curriculum progression framework

- **Pathway:** Foundations → Guided Practice → Applied Projects → Verification
- **SRS cadence:** Day 1, 3, 7, 14, 30 review windows
- **Regression rule:** if mastery score < threshold, return to next-shortest interval

## MSc CS module lanes

| Module | Core outcomes | Assessment mode | Progression stage target |
|---|---|---|---|
| Algorithms & Complexity | Design and analyze performant solutions | Timed coding + oral defense | retained |
| ML/AI Systems | Build and evaluate predictive systems | Lab notebooks + model reports | applied |
| Security & Network Defense | Apply zero-trust and threat mitigation | Incident simulation | applied |
| Data Mining / NLP / CV | Implement domain pipelines | Project-based milestones | verified |

## Planning cost lanes

| Lane | Example drivers | Relative cost |
|---|---|---|
| Platform engineering | API gateway, storage, observability | High |
| Data + AI | vector indexing, inference, model eval | Medium-High |
| Learning UX | assessment tooling, dashboarding, SRS UX | Medium |
| Operations/compliance | security controls, audits, support | Medium |
