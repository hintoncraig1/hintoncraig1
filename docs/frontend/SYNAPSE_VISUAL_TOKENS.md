## Typography

- Display: Inter 700
- Body: Inter 400
- Monospace/code: JetBrains Mono 400

## Component style guidelines

- Cards: dark background + neon border accent by domain color.
- Tables: high-contrast headers, subtle row separators, optional status badges.
- Dashboards: emphasize progression over vanity metrics.
